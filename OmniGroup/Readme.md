OmniGroup的全系列软件注册码生成器，来自[OmniGroup全系列注册机][1]。

支持:

	✅   macOS Sierra 10.12
	✅   OmniGraffle v7.x
	✅   OmniPlan v3.x
	✅   OmniFocus v2.x
	✅   OmniPlan v3.x

打开注册机后，在左下角选择目标软件，然后点击“generate”即可。

亲测OmniGraffle Pro v7可用。
